# Linear algebra lecture 4

The fourth lecture in the linear algebra course, building on the eigenvalue material from [[Linear algebra lecture 3]] to prove the diagonalization theorem and the spectral theorem for symmetric matrices.

## Agenda

1. Diagonalization. When is a matrix $A$ similar to a diagonal matrix $D$? Statement and proof of the theorem: $A$ is diagonalizable if and only if it has $n$ linearly independent eigenvectors.
2. The spectral theorem for symmetric matrices. Every real symmetric matrix is diagonalizable by an orthogonal matrix, and its eigenvalues are real. This is the cleanest case and the one most applications reduce to.
3. Worked examples: diagonalizing a symmetric 3×3 matrix by hand, and the cautionary case of a non-symmetric matrix with repeated eigenvalues that fails to diagonalize.
